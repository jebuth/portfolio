Recent Projects



Cool Space Shooter is a space shooting game that I developed using Python's pyGame API. pygame is an open source python library that allows for multimedia applications built on top of the SDL (Simple DirectMedia Layer) library. The library includes everything needed to create 2-dimensional games such as my space shooter including support for audio, keyboard and mouse events, sprites, and collision detection. In my game, the player controls a mini space rocket/jet-fighter through infinite purple space created by vertically scrolling a surface. The player maneuvers using WASD and fires rockets with spacebar. I chose to create waves of enemies with a "boss" after the 5th wave. I've created animations using multiple sprites for the player's movement (propulsions) and enemy death explosions. I've also added audio clips sounding on certain events such as firing and deaths. pygame provides its own collision detection function but I was unable to get it working so I created a simple one by comparing each objects x,y coordinates in each frame. Working on this application exposed me to video game concepts such as fps(frames per second), game loops, collision detection, and event handling.







Trackr is a cross platform mobile application I've developed using Xamarin.Forms. I decided to use this technology because it allowed me to use Visual Studios which is the best IDE in my opinion. The user interface was implemented using XAML with C# handling the code-behind and SQLite for the lightweight database. The application allows users to create workout routines and save workouts. It helps to track how much weight you've lifted during that workout session for each specific workout and stores the values in a small database along with a timestamp. Once a workout has been logged more than once, the user can then check his/her progress visually by looking at a graph of their completed workouts. The graph was implemented using OxyPlot, which is an open source cross-platform plotting library for .NET. The most difficult part of this project was handling .NET's DateTime object and converting that to a string so that it could be stored in the database. These strings needed to be conerted back to DateTime objects once they were queried to be used as intervals for the x-axis of OxyPlot's line chart.
